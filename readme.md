# YuJiang_tool
## 制作up主,作业大战御酱,风格的视频工具

把```ffmpeg-python```搓了搓做了一个模仿[作业大战御酱](https://space.bilibili.com/351067)风格视频的小工具

思路是先把音乐拼接在一起，然后再根据音乐的时长把图包制作成视频

效果见[重新合成了作业大战御酱的《老师我不想写作业！第十七弹》](https://www.bilibili.com/video/BV1pV4y1g7pi)
